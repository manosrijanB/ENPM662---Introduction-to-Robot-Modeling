This homework assignment for ENPM662 - Fall 2022.​ It involves setting up forward position kinematics for two robots: the Franka Emika Panda and the KUKA LBR iiwa, using the Denavit-Hartenberg (D-H) convention. ​
Position Kinematics - Panda ​

Task: Setup forward position kinematics for the Franka Emika Panda robot. ​
Robot: 7 revolute joints. ​
Steps:

Draw all D-H coordinate frames. ​
Assign frames on corresponding black circles with the right choice of Z axis. ​
Write down the D-H table. ​
Show all successive link transformation matrices. ​
Show the final transformation matrix between the base frame and end-effector frame. ​



Position Kinematics - KUKA ​

Task: Setup forward position kinematics for the KUKA LBR iiwa robot. ​
Robot: 7 DOF. ​
Steps:

Draw all D-H coordinate frames. ​
Write down the D-H table. ​
No need to show homogeneous transformation matrices or perform geometric validation. ​
